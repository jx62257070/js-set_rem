## js-set_rem

### 安装

```
cnpm install js-set_rem
```

### Usage

```javascript
import {setRem} from 'js-set_rem'
/**
 * 根据设备尺寸设置rem所需的font-size	例如:750px->font-size=100px->7.5rem
 * @method setRem
 * @param {Number} 设置最大屏幕(屏幕宽度上限)
 * @param {String} 屏幕宽度为上限时的font-size
 * @return 木得return
*/
setRem(1024,"100")

```
